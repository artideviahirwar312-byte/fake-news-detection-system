# Fake News Detection System

This project is a machine learning based Fake News Detection System. 
It predicts whether a news article is real or fake.

## Features
- Text preprocessing
- Machine learning model for prediction
- User input for checking news

## Technologies Used
- Python
- Machine Learning
- Pandas
- Scikit-learn

## How to Run
1. Install required libraries
2. Run the Python file
3. Enter the news text to check
